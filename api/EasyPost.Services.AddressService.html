All(Nullable<Dictionary<String, Object>>)

List all Address objects.

Declaration
public async Task<AddressCollection> All(Dictionary<string, object>? parameters = null)
Parameters
Type Name Description
System.Nullable<Dictionary<System.String, System.Object>> parameters

Optional dictionary containing parameters to filter the list with. Valid pairs:

  • {"before_id", string} String representing an Address ID. Starts with "adr_". Only retrieve addresses created before this id. Takes precedence over after_id.
  • {"after_id", string} String representing an Address ID. Starts with "adr". Only retrieve addresses created after this id.
  • {"start_datetime", string} ISO 8601 datetime string. Only retrieve addresses created after this datetime.
  • {"end_datetime", string} ISO 8601 datetime string. Only retrieve addresses created before this datetime.
  • {"page_size", int} Max size of list. Default to 20. All invalid keys will be ignored.
Returns
